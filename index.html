App LembraPet

Descrição do Projeto

O LembraPet é um aplicativo mobile desenvolvido em React Native que ajuda usuários a lembrar de vacinas, consultas e cuidados com seus pets. Com ele, é possível:

  • Criar lembretes para vacinas, banho, passeio e outros cuidados.
  • Visualizar lembretes existentes.
  • Marcar lembretes como concluídos.

O app funciona localmente usando dados mock, sem necessidade de backend para a apresentação.

Resumo do Projeto Integrador

O LembraPet é uma solução prática para auxiliar donos de pets na rotina diária, enviando lembretes de vacinas, banho, passeios e cuidados essenciais.

  • Objetivo: Criar um aplicativo mobile intuitivo que organize a rotina dos donos de pets e garanta que nenhum cuidado seja esquecido.
  • Duração do desenvolvimento: 6 meses
  • Valor total estimado: R$ 22.000
