App LembraPet

Descrição do Projeto

O LembraPet é um aplicativo mobile desenvolvido em React Native que ajuda usuários a gerenciar rotinas de cuidados com seus pets, enviando lembretes de vacinas, passeios, banhos e outros cuidados importantes.

O projeto integra funcionalidades como criação de lembretes, histórico de atividades e marcação de tarefas concluídas, tudo funcionando localmente com dados mock, garantindo uma experiência simples, intuitiva e confiável.

Objetivo

Criar um aplicativo mobile intuitivo que:

  • Melhore a organização das rotinas dos donos de pets.
  • Garanta que nenhum cuidado seja esquecido.
  • Ofereça integração futura com backend seguro e escalável.
