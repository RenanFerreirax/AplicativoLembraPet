# LembraPet

## Descrição do Projeto
O LembraPet é um aplicativo mobile desenvolvido em **React Native** que ajuda usuários a lembrar de vacinas, consultas e cuidados com seus pets. Com ele, é possível:

- Criar lembretes para vacinas, passeios, banho e outros cuidados.
- Visualizar lembretes existentes.
- Marcar lembretes como concluídos.

O app funciona localmente usando dados mock, sem backend.

## Tecnologias Utilizadas
- React Native
- Expo
- TypeScript
- Context API (para gerenciar lembretes, tema e autenticação)

---

## Observações
- O app não utiliza backend; todos os dados são armazenados localmente.  
- Funcionalidades principais: criar, editar, remover e concluir lembretes.
