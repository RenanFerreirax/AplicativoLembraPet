Lembra Pet - Expo React Native (TypeScript) skeleton
Instructions:
1. Unzip and open this folder in VS Code.
2. Run: npm install
3. Run: npx expo start
4. For web: npx expo start --web
5. For Android emulator: open AVD, then npx expo start and press "a" or use 'npm run android'

This project is a mock app. Replace mocks with API calls in src/services when ready.
